## JSON 格式

在该项目中，我们使用JSON格式来传输数据。JSON数据的格式如下

```json
{
    "program": [
        {
            "id": "1",
            "name": "program_1",
            "paras":{
                "para1": "value1",
                "para2": "value2"
            }
        },
        {
            "id": "2",
            "name": "program_2",
            "paras":{
                "para1": "value1",
                "para2": "value2",
                "para3": "value3"
            }
        }
    ]

}
```
